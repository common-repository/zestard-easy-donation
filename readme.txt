== Description ==

If you are worried about integrating Donation with WooCommerce? Than yes, we are here to help you. This plugin will helps you to integrate donation on Cart and Checkout page of WooCommerce.

This plugin will help you to integrate donations. That donations can be single as well as multiple. It will allows you to display donations on cart and checkout page with custom settings. By using this, user can do multiple donations also. And after successful order, user will get Donation reciept on their email address.

== Features ==

- Admin can configure Donation view and emails from Admin panel.

- In Email Receipt, admin can setup an email template's status, subject, receipients and additional content from this tab. 

- Admin can add donations on backend. And those donations will be displayed on frontend. It is allows you to add Image and Description of the donation.

- In Backend on Donation page, admin can choose different options like dropdown, pricebar range and input field for selecting donation amount. 

- There is a Track Donation page to view the summary of total donations. It will also allows you apply filter on donation's name and year.

- On Orders page of WooCommerce, admin can check the total donations of current order.

== Installation ==

1. Upload the `zestard-easy-donation` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "Donation Settings" menu to configure the plugin.
4. Once all configuration completes all donations will work according to it.
